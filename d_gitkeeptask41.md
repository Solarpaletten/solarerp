# 📌 ЗАДАЧА 41.1

## Отображение бухгалтерских проводок

После POST:

В документе должен появиться блок:

```
Accounting Entries
```

Таблица:

| Debit | Credit | Amount | Description |
| ----- | ------ | ------ | ----------- |

Данные берём из:

```
journalEntry
journalLines
```

Фильтр по purchaseId.

---

# 📌 ЗАДАЧА 41.2

## Отражение входящего НДС

Для PURCHASE:

Ожидаем проводки:

Пример:

Debit: 2000 (Inventory)
Debit: 1576 (Input VAT 19%)
Credit: 1600 (Accounts Payable)

Проверить:

* создаётся ли строка по VAT
* правильный ли счёт берётся из accountMapping
* сумма VAT корректна

Если нет — исправить Posting Engine.

---

# 📌 ЗАДАЧА 41.3

## Складской приход (Stock Movement View)

После POST должен отображаться блок:

```
Warehouse Receipt
```

Показать:

* Item
* Quantity
* Warehouse
* FIFO lot created
* Date

Данные из:

```
stockMovement
stockLot
```
