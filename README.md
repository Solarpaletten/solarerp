# Solar ERP

## Overview
Solar ERP is a multi-tenant accounting and ERP platform built on a two-level architecture:
- Account Level (Platform)
- Company Level (ERP workspace)

## Architecture
- Next.js 14 (App Router)
- Prisma + PostgreSQL
- Multi-tenant (Tenant → Companies)
- Modular ERP design

## Levels
1. Account Level
   - Dashboard
   - Companies
   - User settings

2. Company Level
   - Company Dashboard
   - Customers
   - Warehouse
   - Bank
   - Documents
   - Reports
